# ✨ AI Social Media Content Generator

&gt; **OptimusAutomate Internship Project** — Create stunning, platform-optimized social media content in seconds using AI.

---

## Overview

This project is an **AI-powered social media content generation tool** built with **Streamlit** and **Groq AI (Llama 3.1)**. It helps users generate engaging social media posts and content calendars for multiple platforms including **LinkedIn**, **Twitter/X**, and **Instagram**.

### Key Features

| Feature | Description |
|---------|-------------|
|**Single Post Generator** | Generate platform-specific posts with custom tone and length |
|**Content Calendar Planner** | Auto-generate multi-day content calendars for your brand |
|**Regenerate Content** | Get fresh variations of generated content with one click |
|**Refine Content** | Edit and improve existing posts with AI assistance |
|**Beautiful UI** | Modern, responsive interface with gradient design |
|**Multi-Platform** | Optimized for LinkedIn, Twitter/X, and Instagram |
